Hot Spots

Went to a new restaurant and want to share how awesome the pizza was? Write a review on Hot Spots! A food review website inspired by Yelp. This site implements a Ruby on Rails backend, PostgreSQL database, React and Redux frontend. Third party APIs include AWS S3, and Google Maps API.

Copters

Ever want to learn how to pilot a helicopter? Now you can while trying to survive in a lava cave with this game, Copters! Steer your vessel through a cave avoiding walls and bricks or else you are toast! This is a pure JavaScript game using only Webpack to manage file dependencies.

Roamy

Out from the bar and want to know which of your favorite late night joints are within 10 minutes? Roamy is here to save the day! This app takes an origin and creates a travel cloud to see where you can go within a set time. This applicatino implements a MERN stack, (MongoDB, Express, React/Redux, Node.js) and Google Maps APIs.
